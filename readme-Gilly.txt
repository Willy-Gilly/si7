C'est un projet Laravel, donc pour le démarrer, il faut lancer "php artisan serve" dans le terminal
à la racine du projet, puis aller à l'adresse 127.0.0.1:8000/ 

(OU déplacer tous les fichiers du dossier public dans le fichier racine, changer le index.php et le fichier .htaccess pour le mettre dans \var\html\ ce qui est nettement plus compliqué à faire en peu de temps.)
